1일반 설정 — 로그인 시 자동 시작(LaunchAtLogin), Dock 아이콘 숨김(LSUIElement). 토글 스위치.
2외관 설정 — 배경 스타일(블러/다크/투명 드롭다운), 아이콘 크기(20~48px 슬라이더), 배경 투명도(50~100%). 실시간 미리보기 반영.
3동작 설정 — 전체화면 Space 자동 숨김, 화면 가장자리 스냅 On/Off.
