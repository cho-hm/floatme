1검색 필드 — 앱 이름으로 필터링. 실시간 검색 (키 입력 시 즉시 필터). 앱이 많을 때 빠르게 찾기 위함.
2이미 추가된 앱 — 플로팅 바에 이미 등록된 앱. 파란 배경 + 체크마크(✓) 표시. 클릭 시 토글(제거).
3미추가 앱 — 플로팅에 등록되지 않은 실행 중 앱. 클릭 시 즉시 플로팅에 추가. 체크마크 없음.
4하단 상태바 — 현재 실행 중인 앱 총 수와 플로팅에 등록된 앱 수 표시.
