1플로팅 바 컨테이너 — 블러 배경(NSVisualEffectView, 기본) 또는 사용자 선택 스타일. 라운드 코너, 드롭 쉐도우. Cmd+드래그로 전체 이동. 화면 가장자리 스냅(20px). 앱 수 증가 시 아이콘 자동 축소(최소 20px).
2앱 아이콘 + 실행 도트 — 실제 앱 아이콘(NSImage) 표시. 하단에 파란 도트로 실행 중 표시. 마우스 호버 시 앱 이름 툴팁. 클릭 시 해당 앱의 모든 윈도우 활성화. Option+클릭 시 즉시 제거.
3현재 포커스 앱 하이라이트 — 현재 최전면 앱에 파란 테두리(2px ring). didActivateApplication 알림으로 실시간 갱신. 도트와 테두리 병행 표시.

1편집 모드 컨테이너 — 0.5초 길게 누르면 진입. 바 테두리가 파란색으로 변경되어 편집 모드임을 표시. 아이콘이 iOS처럼 미세하게 흔들리는 애니메이션. 바 밖 클릭 또는 ESC로 종료.
2제거 버튼 (×) — 각 아이콘 좌상단에 빨간 원형 X 버튼 표시. 클릭 시 축소+페이드아웃 애니메이션으로 즉시 제거. 나머지 아이콘 재정렬. 아이콘 드래그로 순서 변경 가능.
3추가 버튼 (+) — 편집 모드에서만 나타남. 파란 점선 테두리. 클릭 시 앱 선택 팔레트 표시. 메뉴바나 Cmd+Shift+A로도 팔레트 접근 가능.
