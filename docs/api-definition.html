내부 데이터 모델 · v0.1 · 2026-04-17

FloatingApp

플로팅 바에 고정된 개별 앱 정보. Bundle Identifier로 앱을 식별한다.

필드타입필수설명예시
bundleIdentifierStringY앱 고유 식별자 (macOS Bundle ID)"com.apple.dt.Xcode"
appNameStringY앱 표시명 (저장 시점 캐시)"Xcode"
orderIntY플로팅 바 내 표시 순서 (0부터)0

AppSettings

사용자 설정 전체를 담는 루트 모델. UserDefaults에 단일 JSON으로 저장.

필드타입기본값설명
pinnedApps[FloatingApp][]플로팅에 고정된 앱 목록 (순서 보장)
positionBarPosition{100, 100}플로팅 바 좌상단 좌표
orientationBarOrientation.horizontal배치 방향
iconSizeInt36아이콘 크기 (px, 24~48 범위)
backgroundOpacityFloat0.8플로팅 바 배경 투명도 (0.5~1.0)
autoRemoveTerminatedBooltrue종료된 앱 자동 제거 여부
hideInFullscreenBooltrue전체화면 Space에서 숨김 여부
launchAtLoginBoolfalse로그인 시 자동 시작
hideDockIconBoolfalseDock 아이콘 숨김 (메뉴바 전용 모드)

BarPosition

플로팅 바의 화면 좌표. 드래그 종료 시 저장.

필드타입설명
xDouble화면 좌상단 기준 X 좌표 (pt)
yDouble화면 좌상단 기준 Y 좌표 (pt)
